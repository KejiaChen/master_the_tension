# Master the Tension

This is the repository is for reseach article "Master the Tension: Force-Driven Assembly of Deformable Linear Objects with Environmental Contacts".

This repository contains the following contents
- Transition Planning, consisting of [planning workflow](https://github.com/KejiaChen/master_the_tension_transition_plan) and custom [MoveIt 2 Task Constructor](https://github.com/KejiaChen/moveit_task_constructor)
- Adaptive Fixing, consisting of [control framework for Franka Panda robot](https://github.com/KejiaChen/master_the_tension_mios)

The specific workflow and system requirement is specified in each module.
